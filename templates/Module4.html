Purpose of the Physical Layer

Section 4.1 — The foundation of all network communication

4.1.1 Physical Connection

Before anything else...

Before any network communication can occur, a physical connection must be established to a local network — wired or wireless.

NIC = Network Interface Card
Connects your device to the network
Some devices have multiple NICs (wired + wireless)
💡Not all physical connections offer the same level of performance
4.1.2 What the Physical Layer Does

Transports bits across media

  • Accepts a complete frame from the Data Link Layer
  • Encodes it as a series of signals
  • Transmits signals to the local media
  • This is the last step in the encapsulation process
  • The receiving device re-encapsulates and decides what to do

Copper Cabling

Section 4.3 — UTP, STP, and Coaxial

4.3.1 Characteristics

Why Copper?

✅Advantages
Inexpensive · Easy to install · Low resistance to current flow
⚠️Limitations
Attenuation · EMI/RFI interference · Crosstalk
🛡️ Mitigations:
• Attenuation → strict 100m length limit
• EMI/RFI → metallic shielding + grounding
• Crosstalk → twisting wire pairs together
Interactive — Cable Types

Select a cable type to explore:

UTP — Unshielded Twisted Pair

Most common networking media. Terminated with RJ-45 connectors.

🔶 Outer jacket — physical protection
🔄 Twisted pairs — signal protection
🎨 Color-coded insulation — identifies pairs
❌ No shielding
✅Cheapest, easiest to install, most widely used
⚠️Less EMI protection than STP

STP — Shielded Twisted Pair

Better noise protection than UTP. Also uses RJ-45 connectors.

🔶 Outer jacket — physical protection
🛡️ Braided/foil shield — EMI/RFI protection
🛡️ Foil shield per pair — extra protection
🎨 Color-coded insulation
✅Better EMI/RFI protection than UTP
❌More expensive & harder to install than UTP

Coaxial Cable

Used for wireless antenna connections and cable internet.

(outer) Jacket — minor physical protection
🛡️ Woven copper braid — second wire + shield
🔲 Flexible plastic insulation
⚡ Copper conductor — transmits signals
📡Use cases:
Wireless antenna connections
Cable internet installations

UTP Cabling

Section 4.4 — Properties, Standards, Connectors, Cable Types

4.4.1 How UTP Limits Crosstalk

Two Key Properties

1. Cancellation
Each pair: one wire negative, one positive.
Twisted together → magnetic fields cancel
→ Eliminates EMI/RFI
2. Varied twists per foot
Each wire pair twisted differently.
Prevents crosstalk between wire pairs.

Which cable do I need?

🟢 Straight-through

Both ends: same standard (T568A–T568A or T568B–T568B)

Host → Switch
Host → Router

🟠 Crossover

Mixed ends: T568A on one side, T568B on the other

Host → Host
Switch → Switch
Router → Router
Same device types (legacy — most NICs use Auto-MDIX now)

Rollover (Cisco Proprietary)

Host serial port → Router/Switch Console port. Uses adapter. Used for configuration only.

Fiber-Optic Cabling

Section 4.5 — Light-based data transmission

4.5.1 Properties

Why Fiber?

🚀
Highest bandwidth
Up to 100 Gbps+
📏
Longest distance
Up to 100,000 meters
🛡️
Immune to EMI/RFI
100% immune
Made of: flexible, extremely thin strands of very pure glass
Encoding: laser or LED → light pulses = bits
Works like a wave guide — minimal signal loss
Interactive — Single-Mode vs Multi-Mode
🔵 Very small core
🔦 Uses expensive lasers
📏 Long-distance applications
✅ Less dispersion
🟡 Yellow jacket color
🌐Used for long-haul networks, submarine cables, enterprise backbone across buildings
🟠 Larger core
💡 Uses cheaper LEDs
📏 Max 550 meters at 10 Gbps
⚠️ More dispersion
🟠 Orange (or aqua) jacket color
📡Used within enterprise buildings, data centers, short runs

Wireless Media

Section 4.6 — Electromagnetic signals, WLAN, standards

4.6.1 Properties

Wireless carries electromagnetic signals representing bits using radio/microwave frequencies.

✅Greatest mobility — no physical cable needed
📈Wireless connections continue to increase worldwide
📉Coverage area — physical environment impacts signal
🔒Security risk — no physical access required to intercept
🔀Half-duplex — only one device sends/receives at a time

4.6.3 Wireless LAN Requirements
📡

Wireless Access Point (AP)

Concentrates wireless signals from users and connects to existing copper network infrastructure.

💻

Wireless NIC Adapter

Provides wireless communication capability to network hosts (built into most modern devices).

🔒Network admins must apply stringent security policies — WLANs are vulnerable to unauthorized access without physical barriers
