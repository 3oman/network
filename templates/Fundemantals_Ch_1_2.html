Overview of Computing and Information Systems

1
📊 أولاً: البيانات والمعلومات والمعرفة
🔢
البيانات (Data)
المواد الخام
أرقام أو نصوص أو رموز بدون معنى محدد
مثال: الأرقام 3، 6، 9، 12 — لا نعرف ماذا تعني!
لا بد من معالجتها أو إعطائها سياقاً لتصبح مفيدة
💡
المعلومات (Information)
بيانات + معنى
بيانات تمت معالجتها وأصبح لها معنى وقيمة
مثال: 3، 6، 9، 12 = أول أربعة أجوبة في جدول الضرب في 3
البيانات + المعنى = المعلومات
🧠
المعرفة (Knowledge)
معلومات + تطبيق
استخدام المعلومات لحل مشكلة أو اتخاذ قرار
مثال: CGPA = 3 → شركة تعلم أن هذا الطالب مؤهل للعمل
المعلومات + التطبيق = المعرفة

🔧 ثالثاً: مكونات نظام المعلومات الحاسوبي
👥
١. الموارد البشرية (People)
المستخدمون النهائيون (End-Users)يستخدمون النظام: طلاب، موظفون، عملاء
متخصصو نظم المعلوماتيبنون النظام: محللون، مبرمجون، مختبرون
🖨️
٢. الموارد المادية (Hardware)
أجهزة الإدخال:لوحة مفاتيح، ماوس، ماسح ضوئي
المعالج المركزي (CPU):يعالج التعليمات والبيانات
الذاكرة الداخلية (RAM):مؤقتة، تُمسح عند إيقاف التشغيل
الذاكرة الخارجية (HDD):دائمة، تحتفظ بالبيانات
أجهزة الإخراج:شاشة، طابعة
💿
٣. الموارد البرمجية (Software)
برامج النظام (Systems Software) نظام التشغيل برامج التطوير برامج المساعدة
برامج التطبيقات (Application Software) Word / Excel نظام تسجيل الطلاب نظام المكتبة
🗄️
٤. موارد البيانات (Data)
سجلات، ملفات، تقارير، وثائق وقواعد بيانات
مثال: سجل الطالب (رقم الطالب، الاسم، التخصص، البريد الإلكتروني)
🌐
٥. موارد الشبكة (Network)
الخادم (Server):يتحكم بتدفق المعلومات في الشبكة
أجهزة المستخدمين (Clients):الحواسيب والهواتف المتصلة بالشبكة
معالجات الاتصال:هابات، موجهات (Routers)، سويتشات
✅ مزايا الشبكة
مشاركة الأجهزة والموارد
خفض التكاليف
تسريع تدفق المعلومات
❌ عيوب الشبكة
تكلفة التركيب عالية
مخاطر أمنية (اختراق، فيروسات)

👥

🛒
٢. قوة المشترين (Buyers)قوتهم تزيد إذا اشتروا كميات كبيرة أو كانوا مركّزين

🔗
الأنشطة الأولية (Primary Activities)
تخلق القيمة الرئيسية للعميل
اللوجستيات الداخلية:استلام وتخزين المواد الخام
العمليات:تحويل المدخلات لمنتجات نهائية
اللوجستيات الخارجية:معالجة الطلبات والتوزيع
التسويق والمبيعات:التسعير وإدارة القنوات
الخدمة:الصيانة وما بعد البيع
🛠️
الأنشطة الداعمة (Support Activities)
تدعم الأنشطة الأولية
المشتريات (Procurement):شراء المواد الخام والمستلزمات
تطوير التكنولوجيا:البحث والتطوير
الموارد البشرية:التوظيف والتدريب
البنية التحتية:الإدارة العامة، التخطيط، المالية
🗂️ رابعاً: أنواع نظم المعلومات
النوع الاختصار المستوى الإداري الوظيفة الرئيسية مثال
نظام معالجة المعاملات TPS المستوى التشغيلي (أسفل) تسجيل العمليات اليومية الروتينية وتخزينها نظام نقاط البيع، نظام الرواتب، تسجيل الحضور
نظام معلومات الإدارة MIS الإدارة الوسطى ينتج تقارير منظمة ومحددة من قواعد البيانات تقارير المبيعات الشهرية، تقارير الأداء
نظام دعم القرار DSS الإدارة الوسطى/العليا يحلل بيانات ويجيب على أسئلة غير متوقعة وغير متكررة تحليل مبيعات منتج جديد، تحليل "ماذا لو"
نظام معلومات تنفيذية EIS/ESS الإدارة العليا (القمة) يعرض ملخصات ورسوم بيانية للمديرين التنفيذيين لوحة تحكم تشغيلية للمدير التنفيذي
💡 ترتيب الهرم الإداري: TPS (القاعدة — الأكثر تفصيلاً) ← MIS ← DSS ← EIS (القمة — الأكثر تلخيصاً)
